# Introduction

Extract the tfidf features of tags and title.

# Usage

```shell
usage: tf_idf.py [-h] [--data_path DATA_PATH] [--save_title SAVE_TITLE]
                 [--save_tags SAVE_TAGS]

------tf-idf任务-------

optional arguments:
  -h, --help            show this help message and exit
  --data_path DATA_PATH
                        文本json数据的路径
  --save_title SAVE_TITLE
                        存放提取的title特征的路径
  --save_tags SAVE_TAGS
                        存放提取的tags特征的路径
````

Simply run

```shell
sh tfidf.sh
```
